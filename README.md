# Clove Dental Landing Page

A responsive landing page for Clove Dental, built from Figma design using only HTML5, CSS3, and Vanilla JavaScript.

## Project Structure

```
my-landing-page/
├── index.html          # Main HTML file with semantic structure
├── css/
│   └── style.css      # All styling (colors, typography, layout, responsive)
├── js/
│   └── script.js      # Vanilla JavaScript for interactions
├── images/
│   └── (Figma exported assets)
└── README.md          # Project documentation
```

## Features

### Sections Implemented

1. **Header/Navigation** - Logo with CTA buttons (Book Appointment, Phone)
2. **Statistics Bar** - Key metrics (600+ Clinics, 1300+ Dentists, etc.)
3. **Hero Section** - "Painless Root Canal Treatment in Delhi" with dentist image
4. **Form Section** - "Get Expert Dental Guidance in Delhi" with consultation form
5. **Root Canal Treatments** - Two treatment cards with pricing
6. **Transformations** - Before/after patient photos
7. **Why Choose Clove Dental** - Benefits list with image
8. **Testimonials** - Patient testimonial cards
9. **Book Appointment** - Appointment booking form
10. **Safety Section** - Three safety feature cards
11. **Google Reviews** - Review summary and individual reviews
12. **Find Us Across India** - City cards with clinic counts
13. **Expert Guidance & Health Plan** - Dual card section
14. **FAQ Section** - Expandable frequently asked questions
15. **Footer** - Logo, copyright, and navigation links

## Technologies Used

- **HTML5** - Semantic markup
- **CSS3** - Custom styling with CSS variables, Flexbox, and Grid
- **Vanilla JavaScript** - No frameworks or libraries

## Design Specifications

### Color Palette
- Primary Orange: `#F7931E`
- Dark Background: `#000000`
- Light Background: `#FFF8F2`
- Captcha Box: `#5A341F`
- Text Dark: `#2C2C2C`
- Text Light: `#FFFFFF`

### Typography
- Font Family: System fonts (San Francisco, Segoe UI, Roboto, Arial)
- Headings: Bold, varying sizes (28px - 48px)
- Body: Regular weight, 14px - 18px

### Responsive Breakpoints
- Desktop: 1024px+
- Tablet: 768px - 1023px
- Mobile: 480px - 767px
- Small Mobile: < 480px

## JavaScript Functionality

- **FAQ Accordion** - Expandable/collapsible FAQ items
- **Form Validation** - Client-side validation for consultation forms
- **Smooth Scrolling** - Smooth scroll behavior for navigation
- **Scroll Animations** - Fade-in animations on scroll
- **Header Effects** - Shadow on scroll
- **Interactive Elements** - City cards, treatment buttons, etc.

## Browser Support

- Chrome (latest)
- Firefox (latest)
- Safari (latest)
- Edge (latest)

## Setup Instructions

1. Extract all files to a directory
2. Ensure the folder structure matches the project structure above
3. Place all Figma-exported images in the `images/` folder
4. Open `index.html` in a web browser

## Notes

- All images are referenced from the `images/` folder
- Placeholder text and images are used where actual assets are not available
- Forms are set up for client-side validation only (backend integration needed for production)
- The design is fully responsive and tested across different screen sizes
